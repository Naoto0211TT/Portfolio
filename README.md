# Portfolio

転職活動用に制作したポートフォリオサイトです。

フロントエンド開発の経験に加え、
写真活動やデザイン経験など、人柄も伝わる構成を意識して制作しました。

## URL

[Portfolio Site](https://naoto0211tt.github.io/Portfolio/)

## コンセプト

採用担当者が短時間で
「経歴」「スキル」「人物像」を把握できることを目的に制作しました。

## 使用技術

- HTML
- CSS
- JavaScript
- Git / GitHub
- GitHub Pages
- Font Awesome

## 主な実装

- レスポンシブ対応
- スクロールアニメーション
- ハンバーガーメニュー
- 複数ページ構成
- GitHub Pagesによる公開

## 開発手法

生成AIを活用したAI駆動開発で制作しました。
基本構成はAntiGravityをベースに作成し、レイアウトや情報設計、文章、デザインは目的に応じて取捨選択・改善を重ねながら調整しました。また、ChatGPTは実装時の相談やバグ修正、デザイン改善の検討に活用しています。
